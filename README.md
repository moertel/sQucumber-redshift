Define and execute SQL integration tests for AWS Redshift. It's Cucumber - for SQL!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'squcumber-redshift'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install squcumber-redshift
